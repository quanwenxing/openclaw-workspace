# Java 8 / JSF 1.2 → Java 21 / Jakarta Faces 4 キャッチアップ計画

## 目的
- JSF未経験の状態から、7週間でプロジェクト企画・見積・技術的論点整理・移行推進に必要な理解を獲得する。
- 対象は **Java 8 + JSF 1.2 のレガシーWebアプリ** を **Java 21 + Jakarta Faces 4** へ移行する案件。
- 学習だけでなく、**実案件で使う観点（調査項目・リスク・PoC・説明責任）** までカバーする。

## 成果物
- `RESULT.md`: 7週間の詳細ロードマップ本体
- `LOG.md`: 調査ログ・出典メモ
- `META.json`: 作業メタ情報

## 使い方
1. まず `RESULT.md` を読む
2. 週次の学習・調査・PoCをそのまま実行する
3. 実システム固有の差分（使用コンポーネント、アプリサーバ、JSP/Facelets、独自タグ）を追加で棚卸しする
